# 🗺️ User Journey Map — Parcours d'Achat E-Commerce

## Étapes du Parcours

| Étape | Action Utilisateur | Point de Contact | Émotion | Friction |
|-------|-------------------|-----------------|---------|---------|
| 1. Découverte | Arrive via Google / Pub | Page d'accueil | 😊 Curieux | Temps de chargement lent |
| 2. Navigation | Parcourt les catégories | Menu / Filtres | 😐 Neutre | Filtres peu visibles |
| 3. Recherche | Cherche un produit précis | Barre de recherche | 😤 Frustré | Résultats non pertinents |
| 4. Fiche Produit | Consulte les détails | Page produit | 😊 Intéressé | Photos insuffisantes |
| 5. Panier | Ajoute au panier | Bouton CTA | 😊 Motivé | Confirmation peu visible |
| 6. Checkout | Saisit ses informations | Formulaire | 😤 Frustré | Trop de champs obligatoires |
| 7. Paiement | Choisit son moyen de paiement | Page paiement | 😰 Anxieux | Manque de sécurité perçue |
| 8. Confirmation | Reçoit la confirmation | Email / Page | 😊 Soulagé | Email trop tardif |

## Moments Critiques (Drop-off)

- 🔴 **Recherche → Fiche Produit** : 35% d'abandon (résultats irrelevants)
- 🔴 **Checkout → Paiement** : 28% d'abandon (formulaire trop long)
- 🟡 **Panier → Checkout** : 18% d'abandon (frais de livraison cachés)
