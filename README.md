# 🎨 Analyse UX : Optimisation du Parcours d'Achat 2024

> **Stack :** UX Research · Nielsen Heuristics · Python · Plotly · Figma  
> **Type :** UX Analysis · Conversion Optimization · User Journey

---

## 🎯 Objectif

Améliorer l'expérience utilisateur et réduire les points de friction
sur une application e-commerce (navigation, recherche, panier, checkout).

---

## 🔬 Méthodologie

```
1. Cartographie du parcours utilisateur (User Journey Map)
2. Audit heuristique (10 principes de Nielsen)
3. Identification des étapes de friction (funnel d'abandon)
4. Priorisation via matrice Impact / Effort
5. Recommandations orientées conversion
```

---

## 📊 Résultats Clés

| Métrique | Avant | Après (estimé) |
|----------|-------|----------------|
| Taux de conversion | 0.92% | 1.15% (+25%) |
| Abandon panier | 68% | 46% (-22pts) |
| Abandon checkout | 28% | 18% (-10pts) |
| Score NPS | 32 | 47 (+15pts) |
| Temps checkout | 8.5 min | 3.2 min (-62%) |

---

## 🔴 Top Problèmes Identifiés

1. **Formulaire checkout trop long** → 28% d'abandon (Critique)
2. **Frais de livraison cachés** → 18% d'abandon panier
3. **Moteur de recherche peu pertinent** → 35% d'abandon
4. **Absence de validation temps réel** → erreurs de saisie fréquentes
